## 🕯️ 게임 개요

- **게임 장르**: 드로잉 퍼즐 + 액션 어드벤처  
- **플랫폼**: 웹사이트  
- **게임 이름**: Spectral Atelier (스펙트럴 아틀리에)  

**Spectral Atelier란?**  
"유령들의 화실" / "혼령의 작업실" / "영혼의 아틀리에"

---

## 🌙 게임 컨셉

> 유령, 요괴, 신비한 생물들이 사는 세계에서 그림으로 미스터리를 풀어내는 드로잉 퍼즐 게임

- 드로잉 자체가 무기이자 해결책
- 미스터리 + 코믹 분위기
- 신비로운 영혼 세계 배경
