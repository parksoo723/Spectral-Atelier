# 🎨 Spectral Atelier (스펙트럴 아틀리에)

> 드로잉이 무기가 되는 미스터리 퍼즐 어드벤처 게임

---

## 🕯️ 게임 개요

- **게임 장르**: 드로잉 퍼즐 + 액션 어드벤처  
- **플랫폼**: 웹사이트  
- **게임 이름**: Spectral Atelier (스펙트럴 아틀리에)  

**Spectral Atelier란?**  
"유령들의 화실" / "혼령의 작업실" / "영혼의 아틀리에" 라는 의미로  
신비한 존재들이 그림으로 세계를 변화시키는 공간을 뜻합니다.

---

## 🌙 게임 컨셉

**한 줄 설명**  
> 유령, 요괴, 신비한 생물들이 사는 세계에서 그림으로 미스터리를 풀어내는 드로잉 퍼즐 게임

**핵심 컨셉**
- 드로잉 자체가 무기이자 해결책
- 플레이어가 직접 그린 그림으로 퍼즐 해결
- 미스터리 + 코믹 분위기
- 신비로운 영혼 세계 배경

---

## 🎮 주요 기능

### 플레이어 조작
- 키보드 방향키 → 이동
- 마우스 클릭 → 드로잉 및 상호작용

### 게임 오브젝트
- 플레이어 캐릭터
- 유령 및 요괴 적
- 퍼즐 아이템
- 장애물 오브젝트
- 인터랙션 오브젝트

### 점수 / 레벨 시스템
- 점수 누적 방식
- 스테이지 클리어 시스템
- 경험치 기반 진행 (선택 기능)
- 퍼즐 해결 시 보너스 점수
